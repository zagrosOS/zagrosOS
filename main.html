About Zagros-OS

Zagros-OS is a lightweight 32-bit operating system written mostly in assembly language. It is designed to be minimal, fast, and easy to understand — perfect for OS learners and developers.

Features

  • Simple VGA graphics mode
  • Keyboard input support
  • Minimal bootloader and kernel
  • Efficient memory usage (under 100 MB RAM)
  • Open-source and actively developed

Download

Try Zagros-OS today! Run it on your favorite emulator like QEMU or VirtualBox.
