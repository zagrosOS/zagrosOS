👋 Welcome

Zagros-OS is a new 32-bit operating system written in Assembly, built from scratch for learning, speed, and full control.

🧠 About Zagros-OS

This OS is under heavy development. The goal is to create a minimalist but powerful kernel using x86 Assembly and run directly on hardware.

Version dev 0.1 is the first developer preview.

✨ Features

  • 🖥️ VGA Graphics Mode (320x200)
  • 🧾 Simple welcome screen
  • ⌨️ Basic keyboard input (planned)
  • 💾 Bare metal boot from floppy/ISO
  • 🛠️ Written 100% in Assembly
