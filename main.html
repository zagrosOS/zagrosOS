The Zagros-OS User Handbook

Version dev 0.1 — © 2025 Zagros-OS Developers

Welcome to the Zagros-OS user handbook. This guide will help you understand how to set up, boot, and use the Zagros operating system — a minimal assembly-based OS designed for learning and experimentation.

Use the navigation links on the left to jump between sections.

Overview

Zagros-OS is a small 32-bit operating system written primarily in assembly language. It features a simple VGA interface and is designed for low resource consumption and easy modification.

System Requirements

  • Intel/AMD 32-bit or compatible CPU
  • At least 64 MB RAM (recommended 100 MB or less)
  • VGA-compatible display
  • QEMU, VirtualBox, or real PC for testing

Setup

To run Zagros-OS, download the latest floppy disk image or ISO file from the GitHub releases page. You can boot the image in an emulator or write it to physical media.

Booting

Insert the floppy disk or USB key with the Zagros-OS image into your machine and boot from it. On emulators like QEMU, use the following command:

qemu-system-i386 -fda zagrosos.flp

If booting on real hardware, ensure BIOS boot order is set to your floppy/USB device.

Commands

Zagros-OS includes a minimal shell with basic commands:

  • DIR — List files in current directory
  • RUN <filename> — Run a binary program
  • HELP — Display available commands

Programs

Included programs demonstrate simple functionality:

  • EDIT.BIN — Text editor
  • PONG.BIN — Simple Pong game
  • CALC.BAS — Calculator written in BASIC
